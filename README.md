# เกมลูกเต๋าผู้เล่นหลายคน (Python Multiplayer Dice Game) 🎲

เกมลูกเต๋าแบบผลัดกันเล่น (Turn-based) ที่พัฒนาด้วยภาษา Python มาพร้อมกับ Graphical User Interface (GUI) ที่ออกแบบเอง และระบบการซิงค์สถานะเกมแบบเรียลไทม์โดยใช้สถาปัตยกรรมแบบ Client-Server

## 🚀 จุดเด่นของโปรเจกต์
* **สถาปัตยกรรม Client-Server:** จัดการการเชื่อมต่อเครือข่ายและซิงค์ข้อมูลสถานะของเกมระหว่างผู้เล่นหลายคน
* **Custom GUI:** หน้าต่างผู้ใช้งาน (UI) ที่โต้ตอบได้ แสดงชื่อผู้เล่น ผลการทอยลูกเต๋า และสถานะปัจจุบันของเกม
* **ระบบ Turn-Based:** มีระบบจัดการเทิร์นของผู้เล่นที่แม่นยำเพื่อให้การเล่นเกมเป็นไปอย่างถูกต้อง
* 
## ⚙️ วิธีการรันโปรแกรม

1. **Clone repository:**
   ```bash
   git clone https://github.com/NitiphumNam/your-repo-name.git
   cd your-repo-name
   ```

2. **เปิด Server:**
   ```bash
   python server.py
   ```

3. **เปิด Client:** (รันคำสั่งนี้ในหลายๆ Terminal เพื่อจำลองการเข้าเล่นของผู้เล่นหลายคน)
   ```bash
   python client.py
   ```
